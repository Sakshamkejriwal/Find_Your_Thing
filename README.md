# Find Your Thing (FYT)

**Find Your Thing** is a Lost & Found application designed to help students quickly check for their lost belongings. Built with, simplicity, and Rishiverse aesthetics.

## Features

- **Report Found Items**: Easily report items you've found with descriptions, locations, and photos.
- **Smart Search & Filters**: Search for lost items by name or filter through categories like Electronics, Keys, ID Cards, etc.
<!-- - **Image Compression**: High-quality photo uploads that are automatically compressed to save space. -->
- **Simple Dashboard**: Track the items you've reported and manage them (Edit/Delete).
- **Responsive Design**: Optimized for both desktop and mobile use.
- **Mock Persistence**: Uses Browser LocalStorage to keep data safe even after a refresh.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Navigation**: [React Router Dom v7](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Why This Project?

Losing an item on a large campus can be stressful. **Find Your Thing** was created to digitize this process, providing a centralized platform where students can help each other in real-time.

## Future Scope

- **Full-Fledge Backend**: Transition from LocalStorage to a real-time database (Firebase or MongoDB) with a dedicated Node.js/Express backend.
- **User Authentication**: Implement a secure login system so users can have verified profiles and receive direct notifications.
- **Rishiverse Integration**: Integrate the application directly into **Rishiverse**.
- **Lost Item Mapping**: Use campus 2D maps to pinpoint the exact location where an item was found.

## Getting Started

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/sakshamkejriwal/Find_Your_Thing/
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the dev server**:
    ```bash
    npm run dev
    ```
